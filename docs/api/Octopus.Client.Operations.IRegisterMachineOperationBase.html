If a machine with the same name already exists, it won't be overwritten by default (instead, an System.ArgumentException will be thrown). Set this property to true if you do want the existing machine to be overwritten.

CommunicationStyle

The communication style to use with the Tentacle. Allowed values are: TentacleActive, in which case the Tentacle will connect to the Octopus Server for instructions; or, TentaclePassive, in which case the Tentacle will listen for commands from the server (default).
