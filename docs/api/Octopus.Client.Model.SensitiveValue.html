Class SensitiveValue

When reading from the API, only HasValue will be set, depending on whether a server-side value exists. When writing to the API, passing any non-null NewValue will set the value; otherwise, leaving HasValue with the status quo value will cause the server-side value to be unchanged, while setting HasValue to false will clear any value.
