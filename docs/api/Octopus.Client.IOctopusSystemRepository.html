LoadRootDocument()

Gets a document that identifies the Octopus Server (from /api) and provides links to the resources available on the server. Instead of hardcoding paths, clients should use these link properties to traverse the resources on the server. This document is lazily loaded so that it is only requested once for the current IOctopusSystemRepository.
